# Data Import

**Purpose:** Import raw MT data into FFMT.

## Key Features
- Supported formats: EDI, (añade aquí lo que aplique)
- Folder/batch import
- Basic validation & logs

## How to Use
1. Open **FFMT → Data Import**.
2. Click **Browse** and select your data folder.
3. Review the summary and click **Import**.

## Tips
- Keep input files in a local, writable folder.
- Use consistent file names to avoid parsing issues.
